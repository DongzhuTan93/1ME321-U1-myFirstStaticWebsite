Element

Ett HTML element är en byggkloss som tillsammans med andra HTML element bygger upp en strukturerad HTML sida. Ett element består av en start och sluttagg, attribut och ett värde.

Tagg

En tagg är ett specifik typ av element, exempelvis så är en "p tagg" <p> ett element som används för att skapa en paragraf i ett textstycke. En "hr tagg" <hr> skapar en linje i ett textstycke. Med tagg kan man även syfta på det som påbörjar ett html element ("start tagg") < eller "slut tagg" >.

Attribut

Ett attribut är ett värde som vi ger ett HTML element. Exempelvis så det som vi lägger inom href="" ett attribut som visar en länk till en annan resurs. Exempel på attribut:

Attribut för a (länk elementet)

  • download
  • href
  • rel

Attribut för img (bild elementet)

  • alt
  • height
  • src

Attribut för ol (ordered list elementet)

  • reversed
  • start
  • type
