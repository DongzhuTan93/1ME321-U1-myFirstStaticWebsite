Vad är en regel i CSS och vad består den av?

En CSS regel bestämmer hur ett element ska se ut och placeras på sidan. Den består av en selektor och egenskaper. Exempelvis så kan selektorn vara en klass eller ID som man lägger in i sin HTML kod, sedan så har man en referens till denna i sin CSS kod med egenskaperna som ska appliceras för elementet.

Vad menas med kaskad i CSS?

Med kaskad så menas att om samma selektor finns på två platser i samma CSS fil så används bara den som kommer sist.
